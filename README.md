# FiberLink

Professional IT, Network, CCTV, and Fiber Optic solutions.

## Project

This repository contains the FiberLink frontend built with Vite, React, and Tailwind CSS.

## Local development

Requirements: Node.js (16+) and npm.

Quick start:

```sh
# clone the repo
git clone https://github.com/Adamhafi/fiberlink-main.git
cd fiberlink-main

# install dependencies
npm install

# start dev server
npm run dev
```

Build for production:

```sh
npm run build
```

## Technologies

- Vite
- TypeScript
- React
- Tailwind CSS

## Deployment

You can deploy the `dist/` folder to any static hosting provider (Vercel, Netlify, GitHub Pages, etc.).
